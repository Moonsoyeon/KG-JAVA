- 하나의 소스 파일에는 여러 개의 클래스가 정의 될 수 있다.
- 하나의 클래스만 public의 접근 제어자를 사용할 수 있다.
- 소스 파일의 이름과 동일한 이름을 가진 class는 반드시 main()함수를 구현해야 한다.
- main()함수를 가지고 있는 class는 반드시 public 접근 제어자를 사용해야 한다.
- 하나의 명령어의 끝은 ';'으로 종료해야 한다.

- 입력 기능

- 데이터 타입
  - 문자(1byte) : char, byte
    - 문자 데이터 : 한 문자
    - 문자열 데이터 : 두 문자 이상
    
  - 숫자
    - 정수 : int(4byte, 2^31 - 1), short(2byte), long(8byte)
			- 소수점을 갖지 않는 숫자 데이터
	- 실수 : float(4byte), double(8byte)
			- 소수점을 갖는 숫자 데이터
			
  - 논리 : boolean(1byte)
    - true, false
    
- 연산자
  - 산술 연산자, 관계 연산자, 대입 연산자, 논리 연산자
